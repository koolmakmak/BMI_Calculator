# 🏋️ BMI Calculator

A clean and simple **Body Mass Index (BMI) Calculator** mobile application built with **Flutter**. Enter your weight and height to instantly calculate your BMI, see your health category, and get personalized recommendations.

---

## ✨ Features

- **BMI Calculation** — Input your weight (kg) and height (cm) to calculate BMI instantly
- **BMI Category Display** — Shows your health status across 4 categories:
  - 🔵 Underweight — BMI < 18.5
  - 🟢 Normal — BMI 18.5 – 24.9
  - 🟡 Overweight — BMI 25.0 – 29.9
  - 🔴 Obese — BMI ≥ 30.0
- **Health Recommendations** — Personalized advice based on your BMI result

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (3.x or later)
- [Dart SDK](https://dart.dev/get-dart) (comes with Flutter)
- Android Studio / VS Code with Flutter extension
- An Android/iOS emulator or physical device

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/koolmakmak/BMI_Calculator.git
   cd BMI_Calculator/bmi_app
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```

---

## 📐 BMI Formula

BMI is calculated using the following formula:

```
BMI = weight (kg) / height (m)²
```

| Category    | BMI Range      |
|-------------|----------------|
| Underweight | Less than 18.5 |
| Normal      | 18.5 – 24.9    |
| Overweight  | 25.0 – 29.9    |
| Obese       | 30.0 and above |
